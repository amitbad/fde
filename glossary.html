Chapter 10 · Reference

Glossary — Every Term in One Place

A fast-lookup reference for every concept in the hub. Skim it the night before an interview to lock in precise vocabulary — using the right word (e.g. "Exchange" for a shuffle, "SEQUENCE BY" for CDC ordering) signals real hands-on experience.

A–C

ACID Atomicity, Consistency, Isolation, Durability. Delta Lake brings ACID transactions to data-lake files via its transaction log.
Adaptive Query Execution (AQE) Runtime re-optimization of a Spark plan using post-shuffle stats: coalesces partitions, switches joins, splits skew. On by default.
All-Purpose Compute Interactive, shareable clusters for notebooks. Higher DBU rate than Jobs compute.
APPLY CHANGES INTO DLT CDC syntax that merges a change feed into a target, maintaining SCD Type 1/2 with SEQUENCE BY ordering.
Auto Loader (cloudFiles) Incremental cloud-file ingestion with schema inference/evolution and built-in file tracking.
Autotermination Cluster setting that shuts idle compute down after N minutes — a top cost lever.
Bronze / Silver / Gold Medallion layers: raw history / cleansed-conformed / business aggregates.
Catalyst Spark's cost-based optimizer: logical → optimized → physical plan, plus whole-stage codegen.
Checkpoint Durable directory storing a stream's source offsets and state; the basis of exactly-once recovery.
coalesce(n) Reduce partitions without a full shuffle (may leave uneven partitions).
Control Plane Databricks-managed services: UI, job scheduling, metadata. Never touches customer data directly.
Cost of Delay / DBU DBU = Databricks Unit, the normalized unit of compute consumption billed per hour.

D–G

DABs Databricks Asset Bundles — declarative packaging/deploying of jobs, pipelines, and code across environments.
Data Plane Where compute runs and data is processed: Classic (customer VPC) or Serverless (Databricks-managed).
Data Skew Uneven key distribution causing one task/partition to dominate runtime. Fixed with salting, AQE skew join, broadcast.
Delta Lake Open storage layer adding ACID, time travel, schema enforcement, and data skipping to Parquet.
Delta Sharing Open protocol for secure, real-time data sharing without replication, even to non-Databricks recipients.
DLT Delta Live Tables — declarative pipeline framework with dependency inference, expectations, and lineage.
Exchange The physical-plan operator representing a shuffle / stage boundary.
Executor Worker JVM that runs tasks; has fixed cores (slots) and memory.
Expectations DLT declarative data-quality rules: expect / expect_or_drop / expect_or_fail.
Feature Store Governed feature tables ensuring identical train/serve feature computation; offline + online stores.
Governance Centralized control of access, lineage, and audit — delivered by Unity Catalog.

H–P

IaC Infrastructure as Code — declaring infra (Terraform) so environments are reproducible and audited.
Jobs Compute Ephemeral, single-purpose clusters for automated workloads at a much lower DBU rate.
Liquid Clustering Dynamic Delta file organization by query columns; replaces static partitioning + Z-Order.
Materialized View DLT dataset whose full result is (incrementally) recomputed when inputs change; for aggregations/joins.
Metastore Top-level Unity Catalog container (one per region) holding catalogs/schemas/tables.
MLflow Open-source ML lifecycle: Tracking, Models, Registry, Serving.
OPTIMIZE / VACUUM Delta maintenance: compact small files / delete old unreferenced files past retention.
Partition A chunk of data processed by one task on one slot; the unit of parallelism.
Photon Native C++ vectorized execution engine that speeds CPU-bound SQL/ETL.
Predicate Pushdown Pushing filters to the scan so only matching data is read (Parquet/Delta).
Private Link / Endpoint Private cloud networking between planes/storage, bypassing the public internet.

M–P (new)

Mosaic AI Databricks' umbrella AI layer: Foundation Model APIs, Vector Search, AI/BI Genie, Lakehouse Monitoring, Model Serving, and Databricks Apps — all governed by Unity Catalog.
Foundation Model API (FMAPI) Pay-per-token or provisioned-throughput access to curated LLMs (DBRX, Llama, Mistral) and external model routing (OpenAI, Anthropic) via a single UC-governed endpoint.
Vector Search Databricks-managed serverless vector database synced from a Delta table; used for similarity search in RAG applications.
RAG (Retrieval-Augmented Generation) Pattern: retrieve relevant chunks from Vector Search, inject as context into a prompt, then call an LLM for a grounded answer without hallucination.
AI/BI Genie Conversational BI layer — users ask questions in plain English, Genie generates SQL against UC-governed gold tables and returns charts.
Lakehouse Monitoring Automatic statistical profiling and drift detection on Delta tables; produces profile_metrics and drift_metrics analysis tables in UC.
Databricks Apps Native hosting of Streamlit / Gradio / Dash web apps inside Databricks, inheriting UC auth and scaling to zero when idle.
Predictive Optimization Auto-runs OPTIMIZE and VACUUM on UC-managed tables based on observed query patterns; eliminates manual compaction cron jobs.
Lakehouse Federation UC feature enabling direct SQL queries across external databases (PostgreSQL, Snowflake, BigQuery) without ETL — data stays in the source system.
Provisioned Throughput Dedicated Foundation Model serving capacity (tokens/second guaranteed); use when daily volume exceeds ~500K tokens or strict latency SLA is required.

R–Z

repartition(n) Full shuffle to n partitions; rebalances data (fixes skew), can increase or decrease count.
SCD Type 1 / 2 Slowly Changing Dimension: overwrite current (1) vs keep full history with validity windows (2).
Serverless Databricks-managed compute with near-instant start and folded infra cost; no customer VMs to manage.
Shuffle Network redistribution of data by key for wide transformations; the main cost driver.
Skew Join Hint Hint (or AQE) that splits skewed keys so one task doesn't bottleneck a join.
Spill Task overflowing execution memory to local disk; 10–100× slower, visible in Spark UI.
Storage Credential / External Location UC objects that securely delegate cloud storage access via an IAM role/managed identity + authorized path.
Streaming Table DLT dataset that processes each source record once, incrementally (ingestion).
System Tables UC-governed Delta tables (system.billing.*, system.access.audit) for cost and audit analytics.
Unity Catalog (UC) Central governance layer: 3-level namespace, RBAC, lineage, masking, audit, Delta Sharing.
Watermark Max lateness tolerated for event-time aggregation; lets Spark evict old state.
Whole-Stage CodeGen Fusing operators in a stage into one compiled JVM function to remove per-row overhead.
Z-Order Legacy multi-column data-skipping layout; largely superseded by Liquid Clustering.
